# English Lessons Project

## Overview
This project contains English lesson materials for **intermediate level** students (B1-B2) who are **native Brazilian Portuguese (pt-BR) speakers**. The goal is to improve conversational English and practical vocabulary through situational scenarios.

## Target Audience
- **Level:** Intermediate (B1-B2)
- **Native Language:** Brazilian Portuguese (pt-BR)

## Lesson Generation
For creating new lessons, please use the specialized `english-lesson-generator` skill. It contains all the technical standards, content guidelines, and the multi-step workflow required for this project.

### Core Principles
- **Situational Learning:** Lessons are built around real-world contexts.
- **Grammar in Context:** Every lesson introduces a grammar topic applied to the theme.
- **Print-Ready:** All materials are optimized for A4 printing via HTML/Bootstrap.

## Project Structure
- `aula-X-...html`: Lesson files.
- `aula-X-...-answers.html`: Answer keys and audio transcripts.
- `grammar-tracker.md`: Log of covered and suggested grammar topics.
- `sugestions.md`: List of completed and planned lesson themes.
- `.gemini/skills/english-lesson-generator/`: The source of truth for lesson creation logic.

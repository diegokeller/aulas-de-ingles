# Grammar Concepts Tracker

This document tracks the grammar concepts covered in the English lessons and provides suggestions for future topics to ensure a structured progression for intermediate students.

## Covered Concepts

| Concept | Lesson | Status | Description |
|---------|--------|--------|-------------|
| **Ordinal Numbers** | Lesson 1 | ✅ Learned | Used for dates (1st, 2nd, 3rd, etc.). |

| **Noun to Adjective (-y pattern)** | Lesson 2 | ✅ Learned | Transforming weather nouns (Sun -> Sunny). |
| **Telling the Time** | Lesson 2 | ✅ Learned | Using "past", "to", "half past", "quarter to/past". |
